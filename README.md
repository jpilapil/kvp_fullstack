## Key:Value Pairs

KVP is a platform that connects users based on their technological interests. Users can choose from a list of over 1,500 (yes, 1,500) technologies and pair program together.

### Technologies used:

- React.js
- Node.js
- MySQL
- Express
- Bootstrap
- Heroku
